    Dashboard: Posts:
    • As : Sales Agent, who is logged in to Admin Panel
      I Want to: Share a Lead to Social Network manually or automatically
      So that : I am able to track daily reports of my shares in the Admin Panel
      And : monitor the shared Leads over today's status

    • Scenario : Automatic sharing to Social Networks
      Given: Sales Agent has automatically shared a Lead to any of the applicable social networks [iconized]
      When : The Sharing is successfull
      Then : on the dashboard of the Sales Agent, the following information should be updated:
      - Creation Date
      - Social Network
      - Item Price
      - Item Category
      - Product Name
      - Model
      And : the following information should be filled in manually:
      - Link to Post
      - Link to Group
      - Comments
      - Shares
      - Views
      - Likes

    • Scenario : Manually adding Shared Post information to database:
      Given: Sales Agent manually shared a Lead to any of the applicable social networks, he is logged in and is on the dashboard view
      When : Social Network is selected
      Then : The following Post Details should be selected\filled in:
      - Group [if sharing is done to a group]
      - Group link \ name
      - Link to post
      - Link to post
      - Amount of Likes
      - Amount of Views
      - Amount of Shares
      - Amount of Comments
      And : The following Product Details should be selected\filled in:
      - Category
      - Item Name
      - Model
      - Price
