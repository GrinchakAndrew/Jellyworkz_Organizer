    • Machine component (clickable from any location)
      • General Description:
        Component : Machine component
        Contains the following data:
        - name of the subfactory above
        - name of the machine and S\N
        - currently Iconized Status of the machines
        - Image of the machine with enlarge button to full-screen \ viewport and button to minimize to original size
        - Comment textarea with comment from engineer and date of the comment (text is ellipsozed : ... clickable to read more \ 'read more' button)
        - Auto-feedback from user : 'done'.
        - Write feedback : free-text input from user.
        - Comments history button

        Given: User is found on the Machine component
        When: User clicks on the machine image in view
