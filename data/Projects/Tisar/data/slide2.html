Factory screen

  • Factory screen
    • General Description :
      - Name of the main factory
      - Names of ALL the sub-factories
      - View ALL Sub-factories component
      - Sub-factories component (repeated)
      - Notification Center button
      - The name of the factory is not clickable.
      - There is only one plan with sub-factories.

    • Scenario : View all sub-factories
      Given : user is found on the current location
      When : user clicks 'view all sub-factories'
      Then : user is redirected to the following location
      And : user is able to observe the following information on a given machine:
      - Machine component (repeated):
      - serial number,
      - name of the device,
      - status of the device (color-map, icon, text)
